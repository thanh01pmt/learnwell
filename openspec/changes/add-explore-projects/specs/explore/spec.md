# Capability: Explore Projects

## ADDED Requirements

### Requirement: Category Gallery

The Explore Projects page SHALL display a gallery of project categories including: Intro, Game Design, App, STEM&AI, Physics Simulation, Robotics Simulation, and Algorithm.

#### Scenario: Visual Representation

- **WHEN** user visits `/explore/projects`
- **THEN** they see 7 high-quality cards with distinct icons/images and descriptions.

### Requirement: Create Entry Point

The application SHALL provide a prominent "Create Project" or "+" button for logged-in users to access the Project Hub.

#### Scenario: Header Access

- **WHEN** user is logged in
- **THEN** they see a "Create" button in the Top Header.
